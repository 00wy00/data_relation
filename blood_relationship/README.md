### 说明

1. `doc.md` 文件可引入到 mfe-component/doc.config.js 中，可以让 demo 展示在门户网站中。
2. 执行 `npm run dev` 可在本地开发，也可通过启动

### dev

安装依赖
```
tnpm install
```

启动演示项目

```
tnpm run dev
```

演示项目代码在 demo 中

### 应用

可以通过npm包进行引用
```
npm i @tencent/blood-relation-map
```

导入到文件中
```
import BloodRelationMap from '@tencent/blood-relation-map';
```
